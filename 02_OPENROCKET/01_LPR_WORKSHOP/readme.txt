Naming convention: XX_NC_FINS_M_DAY

XX == Serial number (biggest number == newest rocket)

NC == Nose cone, where:  
* CN == Conical,
* HK == Haack, 
* OG == Ogive, 
* PW == Power Series 

FINS == Fine type, where:
* CDEL == Clipped Delta,
* DELT == Delta,
* ELLP == Elliptical,
* SQRE == Square

M == Month the design was last updated

DAY == Day the design was last updated
